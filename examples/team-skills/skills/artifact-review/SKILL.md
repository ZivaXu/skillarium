# Artifact Review

Start from the rendered artifact. Trace the critical path, record observable evidence, and distinguish broken behavior from taste. Prioritize findings that block comprehension, completion, trust, or accessibility.

Verify desktop and mobile output. Keep recommendations tied to a specific observed state rather than generic design advice.
