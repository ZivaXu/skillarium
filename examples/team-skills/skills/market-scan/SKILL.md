# Market Scan

Define the target decision before searching. Compare direct, adjacent, and potential entrants using current primary sources. Use public product documentation and repository evidence from https://github.com when technical implementation details matter.

Separate confirmed facts from inference. End with the smallest test that could disprove the recommended wedge.
